# CodePhantom 🔐

A beginner-friendly file encryption tool using C++ with Phantom Mode — disguising encrypted files as .jpg or .mp3.

## 🔧 Features
- XOR password-based encryption
- Phantom Mode disguise
- Lightweight & offline
- HTML/CSS interface for interaction

## 📁 File Structure
- encrypt.cpp
- decrypt.cpp
- index.html
- style.css
- readme.md
- logs.txt
- assets/
