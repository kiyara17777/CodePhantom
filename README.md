# CodePhantom 🔐💨

**A lightweight C++ encryption tool with Phantom Mode**  
Encrypts files using XOR and disguises them as .jpg/.mp3

## Features
- Offline encryption & decryption
- Phantom Mode disguises file extensions
- Clean HTML/CSS frontend
